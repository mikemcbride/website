I run a Plex server, mostly for audiobooks. If you're here, it's because I'm trying to add you to my server. Congratulations. Here's what you need to do:

1. Create a free <a href="https://plex.tv" target="_blank" rel="noreferer noforward">Plex account</a>.
2. Send me the email you used to sign up (or send me your Plex user name).
3. Download the <a href="https://apps.apple.com/us/app/prologue-audiobook-player/id1459223267" target="_blank">Prologue</a> app on iOS. For Android, I hear <a href="https://play.google.com/store/apps/details?id=local.oss.chronicle" target="_blank">Chronicle</a> is good.
4. Once I add you to the Plex server, you'll get an email about it. In your app, sign in to Plex and you should see the server as an option and the audiobooks should load.
5. Enjoy!
