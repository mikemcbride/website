I run an Audiobookshelf server. If you're here, it's because I'm trying to add you to my server. Congratulations. Here's what you need to do:

1. Contact me so I can add you to my Audiobookshelf.
2. Download the <a href="https://apps.apple.com/us/app/prologue-audiobook-player/id1459223267" target="_blank">Prologue</a> app on iOS. For Android you can use the official <a href="https://play.google.com/store/apps/details?id=com.audiobookshelf.app&hl=en_US" target="_blank">Audiobookshelf</a> app.
3. In Prologue, click in the upper left corner to add a server. Click "Add server". Select "Audiobookshelf". The server address is <code>https://audiobookshelf.mcbrides.us</code>. I will have already given you your username and password.
4. Enjoy the audiobooks!
